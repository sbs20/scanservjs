## UI Fixes

Various interface refinements:

- **Image transformation controls**: Rotate (90° CW/CCW) and flip (horizontal/vertical)
  buttons in the Scan tab toolbar; transformations apply to both preview and final output,
  are highlighted when active, and are reset when the preview is cleared.
- Assorted bug fixes and visual polish to the scan and files views.
